Campaign Copilot prototype
prototype of a conversational copilot for outbound voice campaigns.

The flow:

Ask which campaigns are running or select the suggested prompts → the copilot lists the active ones.
Pick a campaign → it scopes to that campaign.
Ask for the report → it shows the lead funnel (leads → dialled → connected).
Rechurn → it retries the voicemail/busy leads, with a confirmation guardrail and an audit trail.
Mock data only. No backend, no dependencies.

Run it
Open index.html in any browser, or use the live link via GitHub Pages.
